# hash32 [![hash32](https://img.shields.io/badge/hash-v1.0.0-%234bc51d)](https://www.npmjs.com/package/hash32)

## Install

`npm install hash32 --save`

## Usage

ES6 & TypeScript
```javascript
import { hash32 } from "hash32";

hash32();
```

## AMD
```javascript
const { hash32 } = require("hash32");

hash32();
```
